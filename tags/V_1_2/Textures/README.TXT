LIBRARY OF TEXTURES FOR SWEET HOME 3D v 1.2
-------------------------------------------

This archive contains a SH3T file that describes a set of additional Textures for Sweet Home 3D, 
localized in the following languages:
+ Bulgarian
+ Chinese (Simplified)
+ Chinese (Traditional)
+ Czech
+ Dutch
+ English
- Finnish
+ French
+ German
+ Greek
+ Hungarian
+ Italian
+ Japanese
+ Polish
- Portuguese (Brazilian)
+ Portuguese (Portugal)
+ Russian
+ Spanish
+ Swedish
+ Turkish
- Vietnamese

(the minus sign preceding a language indicates that the SH3T file might be incomplete or in the process of being translated) 

Sweet Home 3D is available at http://sweethome3d.sourceforge.net and should be installed first 
to be able to use the SH3T file of this archive.


HOW TO USE THE SH3T FILE
------------------------

Double-click on the SH3T file to install it or choose "Furniture > Import textures library..." 
from the menu of Sweet Home 3D.
This will copy the SH3T file in the textures plug-in folder of Sweet Home 3D. 
The location of this folder depends on your system:
- under Windows Vista/7/8, this folder is 
    C:\Users\user\AppData\Roaming\eTeks\Sweet Home 3D\textures,
- under Windows XP and previous versions of Windows, this folder is 
    C:\Documents and Settings\user\Application Data\eTeks\Sweet Home 3D\textures,
- under Mac OS X, it's the sub folder 
    Library/Application Data/eTeks/Sweet Home 3D/textures of your user folder,
- under Linux and other Unix, it's the sub folder 
    .eteks/sweethome3d/textures of your user folder.
- with the portable version of Sweet Home 3D, this folder is the sub folder named 
    "data/textures" found in the folder where the portable archive was deployed. 

To uninstall a SH3T file, simply remove it from textures plug-in folder and restart Sweet Home 3D.
The textures plug-in folder is also displayed when you click on a SH3F file listed in the "Textures 
libraries" section of the "Libraries in use" pane. The latter pane is displayed by clicking on the 
"Libraries..." button shown at the bottom of Sweet Home 3D About dialog box.

For your information, a SH3T file is a ZIP file containing some image files and their description.
