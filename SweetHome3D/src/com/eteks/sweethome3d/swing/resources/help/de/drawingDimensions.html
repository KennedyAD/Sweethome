     Dimensionen/Bemaßungen zeichnen
 

Um eine Dimension/Bemaßung einzuzeichnen, müssten Sie zuerst im Menü Plan > Bemaßung erstellen oder das Zeichnet Bemaßungen ein -Werkzeug auswählen.

  Zeichnet Bemaßungen ein -Werkzeug

Jede Dimension/Bemaßung wird mit 3 Klicks im Plan erstellt:

  1. Der erste Klick stellt den Startpunkt einer Bemaßung dar,
  2. der zweite Klick den Endpunkt,
  3. und der dritte Klick hilft Ihnen dabei, die Größe der Erweiterungslinien einzustellen, die am Ende jeder Bemaßung eingezeichnet werden.

Zu jeder Zeit können Sie die Escape-Taste drücken um die Erstellung der Bemaßung abzubrechen.
Während dem zweiten Schritt wird die angezeigte Länge permanent aktuell angezeigt.

Um das Zeichnen von Dimensionen/Bemaßungen zu beenden, wählen Sie Plan > Auswählen oder wählen Sie das Auswahl -Werkzeug an.

 Auswahl -Werkzeug

•

Um Bemaßungen präzise einzuzeichnen, benutzen Sie die Ausrichtungslinien, die als Hilfe von der Position des Mauszeigers zur nächstgelegenen Wand oder Möbelstück angezeigt werden.

•

Die Position und Ausrichtung der Längenangabe einer Bemaßungslinie hängt von der relativen Position der Start- und Endpunkte ab, und von der Position der Erweiterungslinien.

•

Um parallele Bemaßungen/Dimensionen einzuzeichnen, ist der Winkel standardmäßig auf ein Vielfaches von 15° eingestellt. Sie können dies kurzfristig ändern, indem Sie während des Einzeichnens die Shift-Taste gedrückt halten, oder indem Sie den sogenannten Magnetismus in den Einstellungen deaktivieren.

•

Ist eine Bemaßung einmal erstellt, können Sie jederzeit deren Position und Größe mit der Maus ändern.

•

Die Länge, die über der Bemaßung angezeigt wird, ist standardmäßig auf eine gültige Maßeinheit in Ihrem Land eingestellt. Sie können jedoch in den Einstellungen jederzeit zwischen Zoll und Zentimetern wechseln.

•

Die Höhe der Schriftart, welche benutzt wird um den Text der Bemaßung anzuzeigen, kann geändert werden, indem Sie die Skalierung des Plans ändern. Dies können Sie über Plan > Zoom in oder Plan > Zoom out bewerkstelligen.

•

Während das Bemaßung erstellen -Werkzeug ausgewählt ist, sind viele andere Werkzeuge und Menüs deaktiviert. Sollten Sie eines dieser Werkzeuge benötigen, wählen Sie das Auswahl -Werkzeug wieder an.
