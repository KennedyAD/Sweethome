Aide Sweet Home 3D > Création de logements, de murs et de meubles
     Impression d'un logement
 

Pour imprimer un logement, choisissez Fichier > Imprimer....
Ce menu affiche le dialogue d'impression de votre système qui vous permet notamment de choisir l'imprimante, les numéros des pages imprimés, la qualité d'impression...

Par défaut, Sweet Home 3D imprime la liste des meubles, le plan et la vue 3D en cours d'un logement, en utilisant la taille de papier, les marges et l'orientation par défaut.
Si vous voulez modifier ce paramétrage, choisissez Fichier > Mise en page....

Dans le panneau de mise en page, vous pouvez modifier la taille et l'orientation du papier en cliquant sur le bouton Format d'impression. Vous pouvez choisir aussi si la liste des meubles, le plan et la vue 3D d'un logement seront imprimés ou non. Si vous ne voulez pas utiliser l'échelle calculée automatiquement pour remplir une page, vous pouvez choisir une autre échelle dans le champ de saisie Echelle.
Vous pouvez aussi choisir un en-tête et/ou un pied de page qui apparaîtra sur chaque page imprimée. Il est possible de saisir un texte libre dans les champs de saisie En-tête et Pied de page ainsi que des variables qui seront remplacées par leur valeur au moment de l'impression. Voici les 8 variables disponibles actuellement :

  • $pageNumber pour le numéro de page,
  • $pageCount pour le nombre total de pages,
  • $planScale pour l'échelle du plan (soit celle calculée automatiquement, ou celle que vous avez saisie dans le champ Echelle),
  • $date pour la date de l'instant où vous lancerez l'impression,
  • $time pour l'heure de l'instant où vous lancerez l'impression,
  • $name pour le nom du logement,
  • $file pour le fichier complet du logement.
  • $level pour le nom du niveau imprimé.

Pour vous éviter de saisir le nom exact d'une variable, utilisez les boutons Variables affichés sous les champs de saisie En-tête et Pied de page. Comme le signe $ est réservé pour les variables, utilisez le code $$ si vous voulez imprimer un signe $ dans le texte en-tête ou en pied de page.

Pour prévisualiser votre mise en page à l'écran, choisissez Fichier > Aperçu avant impression....

Dans le panneau d'aperçu avant impression, vous pouvez voir comment un logement sera imprimé page par page. Pour passer d'une page à l'autre, cliquez sur les flèches en haut du panneau ou utilisez les flèches du clavier.

•

L'impression peut ne pas fonctionner correctement sous Linux. Si c'est le cas, choisissez Fichier > Imprimer en PDF... et imprimer le fichier PDF créé avec les outils de votre système.

•

Il est possible que l'impression de la vue 3D ne soit pas disponible à cause d'une configuration limitée de votre système. Dans ce cas, la boîte à cocher Imprimer la vue 3D sera décochée et désactivée.

•

Si le logement imprimé contient plus d'un niveau, des options supplémentaires apparaissent dans le panneau de mise en page pour déterminer si vous voulez imprimer uniquement le niveau en cours de sélection ou les niveaux de votre choix. La liste des meubles imprimée ne contiendra que les meubles visibles au(x) niveau(x) imprimé(s), sauf si vous choisissez de n'imprimer que la liste des meubles, auquel cas la liste de tous les meubles sera imprimée. La vue 3D imprimée sera celle en cours de visualisation à l'écran.

•

La vue 3D d'un logement est imprimée à une résolution de 150 dpi de telle façon qu'elle remplisse une page, marges exceptées.

•

Pour représenter l'échelle du plan sur papier, vous pouvez aussi dessiner dans le plan une cote mesurant une longueur standard (1 mètre par exemple).

•

La mise en page choisie est enregistrée dans le fichier d'un logement.

•

Le panneau d'aperçu avant impression est redimensionnable.
