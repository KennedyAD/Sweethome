Help di Sweet Home 3D > Creare case, muri e arredamento
     Salvare e aprire una casa
 

Per salvare una casa, scegli File > Salva o File > Salva come....
Ogni file di Sweet Home 3D ha l'estensione .sh3d. Se non digiti questa estensione, Sweet Home 3D la aggiungerà automaticamente al nome.
Dopo il salvataggio della casa, il suo nome appare nel titolo della finestra principale e nel menù File > Apri casa recente.

Puoi inoltre esportare la vista 3D di una casa in formato OBJ scegliendo Vista 3D > Esporta in formato OBJ... dal menù. Questa caratteristica ti lascia scegliere il file OBJ che descriverà tutti gli oggetti della vista 3D, con il suo file MTL e le immagini delle texture memorizzare nella stessa cartella.

Per aprire una casa esistente, fai doppio click sul suo file, scegli File > Apri... o scegli uno dei file tramite il menù File > Apri casa recente.

•

Un file Sweet Home 3D memorizza la piantina di una casa, l'immagine di sfondo che ha importato, le caratteristiche della vista 3D impostate, e tutti i modelli 3D dell'arredamento che hai aggiunto a questa casa. Questa caratteristica ti consente di scambiare il file della tua casa con persone che non hanno lo stesso catalogo di arredamento.

•

Scegli File > Apri casa recente > Cancella menu se vuoi cancellare la lista dei file di Sweet Home 3D files che compaiono nel menù File > Apri casa recente.
