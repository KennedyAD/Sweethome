Ajuda do Sweet Home 3D > Criar casas, paredes e mobília
     Gravar e abrir uma casa
 

Para gravar uma casa, escolha Ficheiro > Gravar ou Ficheiro > Gravar como....
Todos os ficheiros do Sweet Home 3D têm a extensão .sh3d. Se não introduzir esta extensão, o Sweet Home 3D adiciona-a automaticamente ao nome do ficheiro.
Uma vez gravada a casa, o seu nome aparece na barra de título da janela principal e na lista em Ficheiro > Abrir casa recente.

Pode também:

  • exportar a lista de mobília para um ficheiro com o formato CSV (com tabulações a separar os valores), escolhendo Mobília > Exportar para formato CSV...;
  • exportar a planta para um ficheiro com o formato SVG (Scalable Vector Graphics), escolhendo Planta > Exportar para formato SVG...;
  • criar uma imagem em formato .PNG da vista 3D, escolhendo Vista 3D > Criar fotografia...;
  • criar um vídeo com o formato Quicktime .MOV a partir de um caminho na vista 3D, escolhendo Vista 3D > Criar vídeo...;
  • exportar a vista 3D de uma casa para formato .OBJ escolhendo Vista 3D > Exportar para formato OBJ.... Esta funcionalidade permite-lhe escolher qual o ficheiro .OBJ que descreve todos os objectos da vista 3D, juntamente com o seu ficheiro .MTL e imagens das texturas que serão criadas na mesma pasta.

Para abrir uma casa existente, faça duplo clique no nome do ficheiro. Escolha Ficheiro > Abrir... ou escolha um dos ficheiros em Ficheiro > Abrir casa recente.

•

Um ficheiro do Sweet Home 3D armazena a planta de uma casa, a imagem de fundo que importou, as características da Vista 3D que definiu, e todos os modelos tridimensionais da mobília que adicionou a essa casa. Esta funcionalidade permite-lhe partilhar a sua casa com outras pessoas, mesmo que elas não tenham o mesmo catálogo de mobília.

•

Escolha Ficheiro > Gravar e comprimir para gravar a sua casa num formato comprimido totalmente compatível com o formato predefinido do Sweet Home 3D. Esta operação é mais demorada mas cria um ficheiro bastante mais pequeno.

•

Escolha Ficheiro > Abrir casa recente > menu Apagar se quer apagar a lista de ficheiros do Sweet Home 3D que aparecem em Ficheiro > Abrir casa recente.
