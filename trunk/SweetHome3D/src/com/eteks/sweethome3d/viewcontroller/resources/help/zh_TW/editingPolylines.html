     編輯折線及曲線
 

　　選擇他們的平面圖後，您可以用滑鼠編輯折線的位置。
　　要選擇平面圖中的一條折線，請按一下它。若要選擇多個折線或多個物件在平面圖中，選擇在周圍繪製矩形，或 / 和按shift鍵的同時按一下它們。
　　若要移動所選的物件，將滑鼠指標放裡面選中的框之一這些物件然後拖動並刪除它們，或按鍵盤箭頭。

　　當平面圖在選定的一條折線時，你可以移動任一點，出現在每個點的選定折線點指標。

　　當滑鼠指標位於這些指標之一時，它更改，以指示您可以拖放和那點移動它。
　　顏色和折線的風格也由於其窗格中，按兩下它，或者通過選擇後將顯示編輯平面圖→修改折線...後選擇它。

　　在折線的窗格中，你可以改變 所選折線 的厚度，在其兩端、 聯接線的樣式，它的短劃線樣式和它的顏色繪製箭頭。最後的連接樣式下拉清單中選擇允許繪製一條曲線，而不是線。

•

如果你想要修改的一整套的折線樣式，選擇這些折線並選擇平面圖→修改折線...。 從一條折線為在選定的折線組，另一種不同的屬性的值將顯示為空白在折線窗格中。您也可通過將其複製到剪貼簿中複製上其他折線，折線的風格編輯→複製功能表項目，然後選擇編輯→粘貼樣式後選擇您想要修改的折線。

•

如果你想要禁用啟動的磁性為你繪製，禁用磁性在首選項窗格中或移動點時，按住Alt鍵在 Windows、 Mac OS X 或移下的cmd鍵 + Alt鍵在 Linux 的下線的角度。

•

如果在平面圖中所選的物件包含牆、 房間、 傢俱、 尺寸、 折線和文字，將與拖拉或鍵盤箭頭移動所有這些物件。如果你保持Ctrl鍵 (或下 Mac OS X， alt鍵) 時拖動，並將這些物件按下，他們將被覆制。
