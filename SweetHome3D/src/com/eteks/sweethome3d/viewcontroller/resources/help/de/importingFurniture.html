     Importieren von Mobiliar
 

Um ein existierendes 3D-Modell in den Mobiliarkatalog und/oder den Wohnungsplan zu importieren, wählen Sie Mobiliar > Mobiliar importieren… oder ziehen Sie die Datei eines 3D-Modelles per Drag&Drop auf ein Sweet Home 3D-Fesnter.
Diese Aktionen zeigen alle einen Schritt-für-Schritt-Assistenten der Ihnen helfen wird, eine Datei eines 3D-Modelles zu wählen und, falls notwendig, dessen Einstellungen anzupassen.

1.

Nachdem sie den Assistenten über das Menü Mobiliar > Mobiliar importieren… gestartet haben klicken Sie auf Modell wählen… und wählen Sie eine OBJ, DAE, KMZ, 3DS Datei oder eine Zip-Datei die einen der vorgenannten Dateitypen enthält oder ziehen sie diese Datei per Drag&Drop in den Rahmen. Mit der Schaltfläche Model suchen… werden sie auf die Modellseite von Sweet Home 3D geleitet von der sie weitere Modelle herunter laden können. Wenn das Modell geladen ist, klicken Sie auf Weiter ….

2.

Richten Sie das Modell mit den Pfeilschaltflächen aus so dass die Vorderansicht die Vorderseite des Modells anzeigt. Manchmal werden 3D-Modelle inkorrekt gerendert da ihre Rück- und Frontseiten invertiert sind. Ist dies der Fall, aktivieren Sie das Kontrollkästchen Vorder und Rückseite vertauschen.
Dann klicken Sie auf Weiter ….

3.

Ändern Sie, falls notwendig, den Namen, die Größe, Höhe über dem Boden, Farbe und ob es beweglich ist wie beispielsweise eine Tür oder ein Fenster.
Wenn ein 3D-Modell in den Wohnungsplan oder die Mobiliarliste importiert wurde, können Sie ebenfalls das Kontrollkästchen Zum Katalog hinzufügen aktivieren. In diesem Fall wird die Auswahlliste Kategorie aktiv und sie können eine bestehende Kategorie auswählen oder eine neue eingeben. Wenn sie eine neue eingeben, wird diese in den Mobiliarkatalog aufgenommen.
Deaktivieren Sie das Kontrollkästchen Proportionen beibehalten falls Sie die Höhe, Tiefe oder Breite unabhängig voneinander verändern möchten.
Klicken Sie auf die Schaltfläche Farbe löschen wenn Sie die Standardfarben bevorzugen, nachdem Sie bereits die Farbe des Modells geändert haben.
Danach klicken Sie auf Weiter ….

4.

Drehen Sie das Modell mit der Maus um den besten Ansichtspunkt des Modells für das Modellbild zu finden, das im Katalog, der Liste und im Wohnungsplan angezeigt wird. Danach klicken Sie auf Fertig.

Nachdem der Assisten geschlossen ist, taucht das importierte Modell im Mobiliarkatalog und der Mobiliarliste auf, je nach dem welche Optionen Sie gewählt haben.
Sie können es wie jedes andere Mobiliarstück auch bearbeiten
Der Name der importieren Mobiliar erscheint im Mobiliarkatalog in kursiv. Um die Eigenschaften eines importierten Modells zu bearbeiten, klicken Sie doppelt auf es oder wählen es über das Mobiliar > ändern… Menü aus.

•

Das Importieren von 3D-Modellen per Drag&Drop könnte unter Linux unter Umständen nicht funktionieren. Anmerkung: Erfolgreich unter Linux Mint 21.3 Virginia Cinnamon 6.0.4; Linuxkernel 5.15.0-100-generic

•

Soll ein 3D-Modell in 1°-Schritten an Stelle von 90°-Schritten rotiert werden, halten Sie die Umschalttaste gedrückt, während Sie im zweiten Schritt die Pfeiltasten anklicken.

•

Das Textfeld der Auswahlliste Kategorie ist veränderbar für den Fall, dass Sie Ihre eigene Kategorie von Mobiliar erstellen möchten.

•

Ist die Magnetismus-Option aktiviert, können nur Türen und Fenster mit der Maus auf einer Wand abgelegt werden, und nur für diese Mobiliartypen wird eine Öffnung in der Wand erzeugt. Vergessen Sie daher nicht, das Kontrollkästchen Tür oder Fenster zu markieren, wenn Sie dieses Verhalten für ein importiertes Mobiliarstück wünschen.

•

Nur für Treppen wird automatisch eine Öffnung in der Decke und dem Boden von darüberliegenden Niveaus erzeugt. Vergessen Sie daher nicht, das Kontrollkästchen Treppe zu markieren, wenn Sie dieses Verhalten für ein importiertes Mobiliarstück wünschen.

•

Das Importieren eines 3D-Modelles in den Mobiliarkatalog oder in die Mobiliarliste hängt davon ab, welches der beiden Bereiche den Fokus hat während Sie Mobiliar > Mobiliar importieren… klicken. Falls die Mobiliarliste, der Wohnungsplan oder die 3D-Ansicht den Fokus hat wenn Sie dieses Menü auswählen, erscheint das importierte Modell nur im Katalog, wenn Sie im dritten Schritt des Assistenten das Kontrollkästchen Zum Katalog hinzufügen aktiviert haben.

•

Die Maßeinheit die standardmäßig eingestellt ist, hängt von Ihrem Land ab. Sie können im Dialog Einstellungen zwischen Metern, Zentimetern, Millimetern und Zoll wählen.

•

Da der Katalog von allen Sweet Home 3D-Fenstern geteilt wird, können keine Veränderungen an diesem Katalog rückgängig gemacht werden. (z.B. Importe, Modifizierungen, Löschungen).

•

Die Einstellungen von importierten Mobiliar werden automatisch in deren Einstellungen, 3D-Modell und Icon-Dateien von Sweet Home 3D gespeichert.

•

Über den Menüeintrag Mobiliar > Mobiliarbibliothek importieren… können Sie auch einige Bibliotheken mit vorkonfiguriertem Mobiliar, die als SH3F-Dateien auf der Sweet Home 3D-Webseite verfügbar sind, importieren.
