     Drawing dimensions
 

To draw dimensions, you must first choose Plan > Create dimensions or select the Create dimensions tool.

 Create dimensions tool

Each dimension is created in three clicks in the home plan:

  1. the first click points out the start point of the new dimension,
  2. the second click points out its end point,
  3. the third click helps you to point out the size of the extension lines drawn at each end of the created dimension line. If you don't move the mouse between the second and the third click, the new dimension won't have any extension line.

At any time you may press the Escape key to cancel the creation of the current dimension.
During the second step, each mouse move updates in the plan the new dimension size and the length it displays.

To end the drawing of dimensions, choose Plan > Select or select the Select tool.

 Select tool

•

To help you draw dimensions precisely, use alignment lines drawn from the current mouse pointer location to the closest wall or piece of funiture corner.

•

The location and the orientation of the length drawn on a dimension line depends on the relative location of the dimension start and end points, and of the location of its extension lines.

•

To help you drawing parallel dimensions, the angle of dimensions is by default a multiple of 15°. You may cancel this magnetism by holding down the shift key while drawing, or by disabling magnetism in the preferences pane.

•

Once a dimension is created, you may change its location and its size with the mouse.

•

The length displayed in dimensions is expressed by default in the unit of your country. You may switch to Centimeter or Inch in the preferences pane.

•

The height of the font used to draw dimensions text may be increased or decreased by changing the plan scale with Plan > Zoom in or Plan > Zoom out menu items.

•

While Create dimensions tool is selected, many tools and menus are disabled. If you need these tools, select the Select tool again.
