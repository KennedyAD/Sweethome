     绘制折线和曲线
 

　　去绘制折线和曲线，你必须选择 平面图→建立曲线 选单项目或选择建立曲线工具图标。

 建立曲线 工具

　　您可以通过单击新的曲线，然后在每个点的曲线上直到你双击在其终结点或新的起始点在平面中建立新的曲线Escape键。如果最后双击位于起始点上，然后曲线的路径将被关闭，建立的多边形。 Ctrl 键或alt 键( Mac OS X) 在绘制第一段时，如果折线计数至少 3 个点，则会建立一条曲线。
　　折线和曲线仅在该计划中出现，没有代表在 3D 视图中。

各折线、 曲线和多边形

　　若要绘制折线的结束，选择平面→选择或选择选择工具 (或其他工具)。

 选择工具

•

为了说明您精确地绘制折线，请使用工具提示和放大。如果你按下输入键，平面获得焦点时，工具提示中显示的值将成为可编辑的，让您输入用键盘，长度和当前线段的角度 (如果你没有迹象表明折线的第一个点，然而，工具提示将显示该点的坐标)。 使用向上箭号键、向下箭头键或tab键编辑其他值，再按输入键确认您输入的值。如果你想要再次使用鼠标以绘制折线，只应移动到平面图上。

•

绘制的角度在默认情况下是线段的 15 度的倍数。您可以取消这一磁性在首选项窗格 或绘图时，通过按住Alt键在 Windows、 Mac OS X 或cmd下Shift + Alt键下 Linux 下的键。

•

如图所示的最后一个形状在上面的图像曲线折线的计算方式，如果您创建 4 点封闭的广场，生成的曲线将一个圆圈。

•

一旦创建了一条折线，你可以改变它的位置、 它的颜色和它的样式。您还可以更改折线曲线之后与它修改窗格。

•

选择平面图→放大或计划→缩小更改计划规模，或在保持ctrl键的同时向前或向后滚动鼠标滚轮按下 (或在 Mac OS X 下按下cmd键)。

•

虽然选择了创建折线工具，一些工具和菜单被禁用。 如果你需要这些工具，再选择选择工具。
