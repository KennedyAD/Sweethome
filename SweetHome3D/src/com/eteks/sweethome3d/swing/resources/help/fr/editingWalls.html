     Modification des murs
 

Vous pouvez modifier la position et la longueur des murs d'un logement, soit avec la souris, soit avec le menu Plan > Modifier les murs....
Pour modifier des murs à la souris, sélectionnez les dans le plan du logement. Pour sélectionner un mur dans le plan, cliquez sur celui-ci. Pour sélectionner plusieurs murs dans le plan, dessinez un rectangle de sélection qui les contient, ou cliquez sur les murs en maintenant la touche majuscule enfoncée.
Pour déplacer les murs sélectionnés, positionnez la souris à l'intérieur de la boîte de sélection d'un de ces murs puis glissez-déposez les, ou appuyez sur les flèches du clavier.

Quand un mur est sélectionné dans le plan, vous pouvez aussi déplacer l'une de ses extrémités, avec l'indicateur de taille qui apparaît à chaque extrémité du mur sélectionné.

1.

L'indicateur de taille du point de départ montre le point que vous pouvez glisser-déposer pour déplacer le point de départ du mur.

2.

L'indicateur de départ du mur indique à partir de quel point commence un mur.

3.

L'indicateur d'arrivée du mur indique à quel point se termine un mur.

4.

L'indicateur de taille du point d'arrivée montre le point que vous pouvez glisser-déposer pour déplacer le point d'arrivée du mur.

Quand le pointeur de la souris est sur le point de départ ou d'arrivée du mur sélectionné, celui-ci change d'aspect pour indiquer que vous pouvez glisser-déposer ce point pour le déplacer. Pendant que vous appuyez sur le bouton de la souris, une info-bulle indique la longueur du mur.
Les murs modifiés avec la souris sont dessinés simultanément dans le plan et la vue 3D.

Un mur peut être aussi modifié grâce à son panneau, en double-cliquant sur ce mur dans le plan du logement, ou en choisissant Plan > Modifier les murs... après l'avoir sélectionné.

Dans le panneau du mur, vous pouvez modifier les coordonnées de ses points de départ et d'arrivée, les couleurs des ses côtés gauche et droit, son épaisseur et sa hauteur.
Utilisez les indicateurs de départ et d'arrivée dessinés dans le plan sur chaque mur sélectionné pour distinguer son côté gauche de son côté droit.

•

Si vous voulez modifier les couleurs, l'épaisseur ou la hauteur d'un ensemble de murs, sélectionnez ces murs et choisissez Plan > Modifier les murs.... Les valeurs des attributs différentes d'un mur à l'autre dans l'ensemble des murs sélectionnés, apparaîtront vides dans le panneau des murs.

•

Si vous voulez positionner des couleurs différentes sur le côté d'un mur partagé par des pièces adjacentes, divisez le en plusieurs segments à sa création.

•

Si les éléments sélectionnés dans le plan contiennent des murs et des meubles, tous ces éléments seront déplacés avec un glisser-déposer ou avec les flèches du clavier.

•

L'unité affichée par défaut dans l'info-bulle de longueur et le panneau d'un mur dépend de votre pays. Vous pouvez choisir entre le Centimètre et le Pouce dans le panneau des préférences.

•

Choisissez Plan > Zoom avant ou Plan > Zoom arrière pour modifier l'échelle dans le plan.
