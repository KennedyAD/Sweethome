     Polylinien und Kurven bearbeiten
 

Sie können die Position von Polylinien mit der Maus bearbeiten, nachdem Sie sie im Wohnungsplan ausgewhält haben.
Um eine Polylinie im Wohnungsplan auszuwählen, klicken Sie sie an. Um mehrere Polylinien oder mehrere Objekte im Wohnungsplan auszuwählen, ziehen Sie ein Auswahlrechteck um sie herum, und/oder klicken Sie sie an, während Sie die Umschalttaste gedrückt halten.
Um die ausgewählten Objekte zu bewegen, bewegen Sie den Mauszeiger in das ausgewählte Rechteck eines dieser Objekte und ziehen Sie die Objekte bei gedrückter Maustaste. Alternativ können Sie dazu auch die Pfeiltasten einsetzen.

Ist im Wohnungsplan eine Polylinie ausgewählt, können Sie all ihre Punkte mit den Punktmarkierungen, die an jedem Punkt der ausgewählten Polylinie erscheinen bewegen.

Wenn der Mauszeiger sich über einer dieser Markierungen befindet, ändert er seine Form, um darauf hinzuweisen, dass Sie den Punkt verschieben können.
Farbe und Stil der Polylinie können auch im Dialog Polylinien bearbeiten angepasst werden, der nach einem Doppelklick auf die Polylinie erscheint. Der Dialog Polylinien bearbeiten kann nach Auswahl der Polylinie auch über Plan > Polylinien editieren… aufgerufen werden.

Im Dialog Polylinien bearbeiten können Sie die Dicke der ausgewählten Polylinie, die Pfeile, die an ihrem Ende gezeichnet werden, den Verbinderstil zwischen Liniensegmenten, den Linienstil sowie die Farbe ändern. Die letzte Option in der Auswahlliste Verbinderstil erlaubt es auch, eine Kurve an Stelle mehrerer Linien zu zeichnen.

•

Möchten Sie den Stil mehrerer Polylinien zugleich bearbeiten, wählen Sie diese Polylinien aus und klicken Sie auf Plan > Polylinien editieren…. Attributwerte, die sich zwischen Polylinien in dieser Auswahl unterscheiden, werden im Dialog Polylinien bearbeiten als leere Felder angezeigt. Sie können den Stil einer Polylinie auch auf andere Polylinien übertragen, indem Sie ihn mittels Bearbeiten > Kopieren in die Zwischenablage kopieren und dann, nachdem Sie die zu bearbeitenden Polylinien ausgewählt haben, Bearbeiten > Stil einfügen auswählen.

•

Die Einstellungen zum Magnetismus der Winkel können vorübergehend während Sie Polylinien zeichnen umgeschaltet werden, indem man unter Windows die Alt-Taste, unter macOS die cmd-Taste und unter Linux gleichzeitig die Umschalttaste und die Alt-Taste drücken. Er kann auch in der Werkzeugleiste mit der Schaltfläche Magnetismus aktviert und deaktivert werden. Oder Sie können den Magnetismus permanent deaktivieren, indem Sie dies im Dialog Einstellungen deaktivieren.

•

Befinden sich unter den im Wohnungsplan ausgewählten Objekten Wände, Räume, Mobiliarstücke, Bemaßungen, Polylinien oder Texte, werden all diese Objekte durch Klicken und Ziehen oder mit den Pfeiltasten verschoben. Wenn Sie die Strg-Taste (oder unter macOS die alt / opt-Taste) gedrückt halten, während Sie Elemente ziehen, werden die Elemente dupliziert.
