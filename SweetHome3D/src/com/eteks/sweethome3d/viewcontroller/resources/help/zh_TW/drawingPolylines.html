     繪製折線和曲線
 

　　去繪製折線和曲線，你必須選擇 平面圖→建立曲線 選單項目或選擇建立曲線工具圖示。

 建立曲線工具

　　您可以通過按一下新的曲線，然後在每個點的曲線上直到你按兩下在其終結點或新的起始點在平面中建立新的曲線Escape鍵。如果最後按兩下位於起始點上，然後曲線的路徑將被關閉，建立的多邊形。 Ctrl 鍵或alt 鍵( Mac OS X) 在繪製第一段時，如果折線計數至少 3 個點，則會建立一條曲線。
　　折線和曲線僅在該計畫中出現，沒有代表在 3D 視圖中。

各折線、 曲線和多邊形

　　若要繪製折線的結束，選擇平面→選擇或選擇選擇工具 (或其他工具)。

 選擇工具

•

為了説明您精確地繪製折線，請使用工具提示和放大。如果你按下輸入鍵，平面獲得焦點時，工具提示中顯示的值將成為可編輯的，讓您輸入用鍵盤，長度和當前線段的角度 (如果你沒有跡象表明折線的第一個點，然而，工具提示將顯示該點的座標)。 使用向上箭號鍵、向下方向鍵或tab鍵編輯其他值，再按輸入鍵確認您輸入的值。如果你想要再次使用滑鼠以繪製折線，只應移動到平面圖上。

•

繪製的角度在預設情況下是線段的 15 度的倍數。您可以取消這一磁性在首選項窗格 或繪圖時，通過按住Alt鍵在 Windows、 Mac OS X 或cmd下Shift + Alt鍵下 Linux 下的鍵。

•

如圖所示的最後一個形狀在上面的圖像曲線折線的計算方式，如果您創建 4 點封閉的廣場，生成的曲線將一個圓圈。

•

一旦創建了一條折線，你可以改變它的位置、 它的顏色和它的樣式。您還可以更改折線曲線之後與它修改窗格。

•

選擇平面圖→放大或計畫→縮小更改計畫規模，或在保持Ctrl鍵的同時向前或向後滾動滑鼠滾輪按下 (或在 Mac OS X 下按下cmd鍵)。

•

雖然選擇了創建折線工具，一些工具和功能表被禁用。 如果你需要這些工具，再選擇選擇工具。
