     Нанесение размеров
 

Вы можете изменить положение и размер размера с помощью мышки.
Для начала выделите размер. Чтобы выделить несколько размеров, выделите их с помошьью прямоугольной области или выделяйте по отдеьлности с зажатой клавишей SHIFT.
Переместить выделенное можно с помошью стрелок клавиатуры или двигая выделенное мышью.

Когда одно измерение выделено на плане, вы можете также :

  • переместить точки старта и конца разметки, и индикатор будет показывать актуальное значение при изменении длины размерной линии

1.

The start point size indicator shows the point and the extension line you can drag and drop to move the start point of the dimension.

2.

The extension lines size indicator shows the point on the dimension line you can drag and drop to resize its extension lines.

3.

The end point size indicator shows the point and the extension line you can drag and drop to move the end point of the dimension.

When the mouse pointer is upon on of these indicators, it changes to indicate you can drag and drop that point to change the matching attribute of the selected dimension. While you drag and drop the mouse, the size the modified dimension and the length it displays is updated in plan.

•

When only one dimension is selected, you may move it only if the initial location of the mouse pointer is on the dimension line and not on the extension lines size indicator.

•

To help you keep dimensions parallelism, the angle of a modified dimension is by default a multiple of 15°. You may cancel this magnetism by holding the shift key pressed during mouse move, or by disabling magnetism in the preferences pane.

•

If plan selected items contain walls, furniture and dimensions, all these items will be moved with a drag and drop or the keyboard arrows.

•

The length displayed in dimensions is expressed by default in the unit of your country. You may switch to Centimeter or Inch in the preferences pane.

•

The height of the font used to draw dimensions text may be increased or decreased by changing plan scale with Plan > Zoom in or Plan > Zoom out menu items.
